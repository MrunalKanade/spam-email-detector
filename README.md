\# 📧 Email Spam Detection using Machine Learning



\## 🚀 Overview

This project builds a machine learning model to classify emails as Spam or Not Spam.



\## 🛠 Tech Stack

\- Python

\- Scikit-learn

\- NLP (TF-IDF)



\## 📊 Model Used

\- Logistic Regression

\- Naive Bayes



\## 📈 Accuracy

Achieved ~97% accuracy on test data.



\## 🔥 Features

\- Text preprocessing

\- Feature extraction using TF-IDF

\- Multiple model comparison

\- Real-time prediction
